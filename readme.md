# Шпаргалка по командам GIT

|Команда|Описание|Комментарий|
|----------------|---|---|
|`git init`|инициализация репы в папке|для отмены:  `rm -rf .git`|
|`git status`|выводит статусы  - ветку  - коммиты  - что попадёт в коммит||
|`git add %имя файла%`|добавляет файл в коммит||
|`gid add .`|добавляет в коммит все изменения в папках||
|`git commit`|коммитит изменения. ключ `-m` добавляет комментарий||
|`git log`|выводит историю коммитов||
|`git remote`|для работы с удалённой репой||
|`git remote add`|добавляет удалённую репу, связывая с локальной.  синтаксис:  `git remote add %имя_репы% %URL%`||
|`git remote -v`|выводит вербозный список удаленных реп||
|`git push`|пушит коммит на удаленную репу.  для первого пуша нужно указать имена удаленной репы и текущей ветки, например:  `git push -u origin master`||
